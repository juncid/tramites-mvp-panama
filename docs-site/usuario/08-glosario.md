# 8. Glosario

Definiciones de términos técnicos y conceptos utilizados en el sistema.

---

## A

**Acceso Directo** _(Shortcut)_
: Enlace rápido a una función o página específica del sistema. En el Panel de Control encontrarás accesos directos a funciones frecuentes.

**Actualización de Datos** _(Data Update)_
: Proceso de modificar información personal o de trámites en el sistema. Algunas actualizaciones requieren verificación adicional.

**Adjuntar** _(Attach)_
: Acción de añadir archivos (documentos, imágenes) a una solicitud o trámite. Ejemplo: "Adjuntar pasaporte en formato PDF".

**Agente** _(Agent)_
: Representante del Servicio Nacional de Migración que puede actuar en nombre de un usuario. Requiere autorización formal.

**Alerta** _(Alert)_
: Notificación importante del sistema que requiere atención inmediata. Aparece como banner o pop-up en pantalla.

**Antecedentes Penales** _(Criminal Record Certificate)_
: Documento oficial que certifica la presencia o ausencia de registros penales de una persona. Requerido para la mayoría de trámites migratorios.

**API** _(Application Programming Interface)_
: Interfaz de programación de aplicaciones que permite la comunicación entre diferentes sistemas de software.

**Apostilla** _(Apostille)_
: Certificación internacional que valida documentos públicos extranjeros según el Convenio de La Haya. Requerida para documentos de países firmantes.

**Aprobación** _(Approval)_
: Decisión favorable sobre una solicitud o trámite. Estado final positivo en el ciclo de vida del trámite.

**Archivo Adjunto** _(Attachment)_
: Documento digital anexado a una solicitud. Formatos aceptados: PDF, JPG, PNG.

**Auditoría** _(Audit)_
: Registro de todas las acciones realizadas en el sistema para fines de seguridad y trazabilidad.

**Autenticación** _(Authentication)_
: Proceso de verificar la identidad de un usuario mediante credenciales (correo y contraseña).

**Autorización** _(Authorization)_
: Permiso para acceder a recursos o funciones específicas del sistema basado en el rol del usuario.

---

## B

**Backup** _(Backup)_
: Copia de seguridad de la información del sistema realizada periódicamente para prevenir pérdida de datos.

**Banner** _(Banner)_
: Mensaje informativo que aparece en la parte superior del sistema, generalmente para anuncios importantes o mantenimientos.

**Base de Datos** _(Database)_
: Sistema donde se almacena toda la información de trámites, usuarios y documentos de forma segura y organizada.

**Bot** _(Bot)_
: Programa automatizado que responde consultas frecuentes en el chat fuera del horario laboral.

**Browser** _(Navegador)_
: Programa utilizado para acceder al sistema web (Chrome, Firefox, Safari, Edge).

---

## C

**Caché** _(Cache)_
: Almacenamiento temporal de datos en el navegador para mejorar la velocidad de carga. Limpiar el caché puede resolver problemas de visualización.

**Campo Obligatorio** _(Required Field)_
: Campo de formulario que debe completarse para poder continuar, generalmente marcado con asterisco (*).

**Cancelación** _(Cancellation)_
: Acción de anular o detener un trámite en proceso. Los pagos no son reembolsables tras cancelación.

**Carné de Residente** _(Resident Card)_
: Documento oficial físico que acredita tu estatus de residencia legal en Panamá.

**Carné Provisional** _(Provisional Card)_
: Documento temporal emitido mientras se procesa el carné definitivo. Válido por 30-90 días.

**Catálogo** _(Catalog)_
: Lista predefinida de opciones en el sistema (ej: tipos de trámites, nacionalidades, causas humanitarias).

**Certificado Digital** _(Digital Certificate)_
: Documento electrónico que garantiza la autenticidad e integridad de comunicaciones en línea.

**Certificado Médico** _(Medical Certificate)_
: Documento que acredita el estado de salud de un solicitante. Debe ser emitido por centro médico autorizado.

**Chat en Vivo** _(Live Chat)_
: Herramienta de mensajería instantánea para comunicación con soporte en tiempo real.

**Cita** _(Appointment)_
: Reunión programada en oficina de migración para entrevistas, firma de documentos o recogida de carnés.

**Código de Verificación** _(Verification Code)_
: Código único asignado a cada trámite para verificar autenticidad y permitir seguimiento.

**Comprobante** _(Receipt)_
: Documento que certifica el pago realizado o la recepción de una solicitud.

**Conflicto Armado** _(Armed Conflict)_
: Una de las causas humanitarias para solicitar PPSH. Situación de guerra o violencia generalizada en el país de origen.

**Cookies** _(Cookies)_
: Pequeños archivos almacenados en tu navegador que permiten recordar preferencias y mantener sesión activa.

---

## D

**Dashboard** _(Panel de Control)_
: Página principal que muestra un resumen visual de todos tus trámites, notificaciones y acciones rápidas.

**Dato Sensible** _(Sensitive Data)_
: Información personal protegida por leyes de privacidad (ej: número de documento, datos médicos).

**Declaración Jurada** _(Sworn Statement)_
: Documento legal donde el solicitante declara bajo juramento que la información proporcionada es verdadera.

**Decreto Ejecutivo** _(Executive Decree)_
: Normativa emitida por el poder ejecutivo que regula procedimientos migratorios.

**Desastre Natural** _(Natural Disaster)_
: Causa humanitaria para PPSH relacionada con catástrofes naturales (terremotos, huracanes, inundaciones).

**Documento Adicional** _(Additional Document)_
: Archivo suplementario que puede agregarse a un trámite existente cuando el funcionario lo solicita.

**Documento Original** _(Original Document)_
: Documento físico emitido por autoridad competente, no copia ni fotocopia (salvo que esté certificada).

**DPI** _(Dots Per Inch)_
: Unidad de resolución de imagen. Mínimo requerido: 300 DPI para documentos escaneados.

**Duplicado** _(Duplicate)_
: Copia oficial de un documento perdido o dañado. Puede solicitarse en caso de pérdida del carné de residente.

---

## E

**Edición** _(Edit)_
: Modificación de información previamente ingresada. Solo disponible en ciertos estados del trámite.

**Encriptación** _(Encryption)_
: Proceso de codificación de datos para protegerlos durante transmisión y almacenamiento.

**Entrevista Personal** _(Personal Interview)_
: Reunión obligatoria con funcionario de migración como parte del proceso de evaluación de ciertos trámites (especialmente PPSH).

**Estado Civil** _(Marital Status)_
: Situación legal respecto al matrimonio (soltero, casado, divorciado, viudo, unión libre).

**Estado del Trámite** _(Application Status)_
: Indica la fase actual del proceso de un trámite (Recibido, En Revisión, Aprobado, etc.).

**Etapa** _(Stage)_
: Cada una de las fases secuenciales que componen un workflow. Debe completarse antes de avanzar a la siguiente.

**Expediente** _(File)_
: Conjunto completo de documentos y datos relacionados con un trámite específico.

---

## F

**Filtro** _(Filter)_
: Herramienta para refinar búsquedas y listas según criterios específicos (fecha, estado, tipo).

**Firma Digital** _(Digital Signature)_
: Método electrónico de firmar documentos que tiene validez legal.

**Formulario** _(Form)_
: Conjunto estructurado de campos que deben completarse para ingresar información al sistema.

**Funcionario** _(Officer)_
: Empleado del Servicio Nacional de Migración autorizado para revisar y aprobar trámites.

---

## G

**Gestión de Documentos** _(Document Management)_
: Sistema para organizar, almacenar y recuperar documentos digitales asociados a trámites.

**Guía de Usuario** _(User Guide)_
: Documento (este manual) que explica cómo usar el sistema paso a paso.

---

## H

**Historial** _(History)_
: Registro cronológico de todas las acciones y cambios realizados en un trámite.

**Horario Laboral** _(Business Hours)_
: Lunes a Viernes, 8:00 AM - 4:00 PM (horario de atención de soporte y oficinas).

---

## I

**Inactividad** _(Inactivity)_
: Período sin interacción con el sistema. Por seguridad, la sesión se cierra tras 30 minutos de inactividad.

**Incidente** _(Incident)_
: Problema o error en el sistema reportado para resolución por equipo técnico.

**Inicio de Sesión** _(Login)_
: Proceso de acceder al sistema mediante credenciales válidas (correo y contraseña).

---

## J

**JPG/JPEG** _(JPG/JPEG Format)_
: Formato de imagen comprimida. Uno de los formatos aceptados para fotografías y documentos escaneados.

---

## L

**Legalización** _(Legalization)_
: Proceso de autenticación de documentos extranjeros mediante cadena de certificaciones consulares. Alternativa a la apostilla.

**Línea de Tiempo** _(Timeline)_
: Visualización cronológica del progreso de un trámite mostrando etapas completadas y pendientes.

**Log** _(Log)_
: Registro automático de eventos del sistema para auditoría y depuración.

---

## M

**Mantenimiento** _(Maintenance)_
: Período programado donde el sistema no está disponible para actualizaciones o reparaciones. Generalmente domingos de 1-5 AM.

**MB** _(Megabyte)_
: Unidad de medida de tamaño de archivo. Límite máximo por documento: 5 MB.

**Menú** _(Menu)_
: Barra de navegación con opciones para acceder a diferentes secciones del sistema.

**Migración** _(Migration)_
: Movimiento de personas de un país a otro. También se refiere al Servicio Nacional de Migración.

**MITRADEL** _(Ministerio de Trabajo y Desarrollo Laboral)_
: Entidad gubernamental que emite permisos de trabajo en Panamá.

---

## N

**Nacionalidad** _(Nationality)_
: País al cual una persona pertenece por nacimiento o naturalización.

**Naturalización** _(Naturalization)_
: Proceso legal mediante el cual un extranjero obtiene la ciudadanía panameña.

**Nivel de Acceso** _(Access Level)_
: Conjunto de permisos asignados a un usuario basado en su rol (usuario, funcionario, administrador).

**Notificación** _(Notification)_
: Alerta automática enviada al usuario sobre eventos importantes en sus trámites (por email, SMS, o dentro del sistema).

**Número de Referencia** _(Reference Number)_
: Código único asignado a cada trámite para identificación y seguimiento (ej: TRAM-2025-0001234).

---

## O

**Oficina Regional** _(Regional Office)_
: Sucursal del Servicio Nacional de Migración en provincias fuera de la Ciudad de Panamá.

**ONPAR** _(Oficina Nacional para la Atención de Refugiados)_
: Entidad que procesa solicitudes de refugio en Panamá.

---

## P

**Panel de Control** _(Dashboard)_
: Ver "Dashboard".

**Pasaporte** _(Passport)_
: Documento de viaje internacional. Generalmente requerido para trámites migratorios.

**Pago** _(Payment)_
: Transferencia monetaria requerida para procesar ciertos trámites. Métodos: tarjeta, transferencia, efectivo.

**Perfil** _(Profile)_
: Página donde se visualiza y edita información personal del usuario.

**Permiso de Trabajo** _(Work Permit)_
: Autorización legal para ejercer actividades laborales en Panamá.

**Persecución Política** _(Political Persecution)_
: Causa humanitaria para PPSH relacionada con amenazas por opiniones o actividades políticas.

**PIN** _(Personal Identification Number)_
: Código de seguridad personal (en este contexto, puede referirse a código de verificación).

**PNG** _(Portable Network Graphics)_
: Formato de imagen sin pérdida de calidad. Aceptado para documentos y fotografías.

**PPSH** _(Permiso Por razones de Seguridad Humanitaria)_
: Permiso especial otorgado a personas en situaciones de vulnerabilidad humanitaria. Vigencia: 1 año renovable.

**Pregunta Frecuente** _(FAQ - Frequently Asked Question)_
: Consulta común respondida en la sección de ayuda del sistema.

**Prórroga** _(Extension)_
: Extensión del tiempo permitido de estadía en el país.

---

## Q

**Queja** _(Complaint)_
: Reporte formal de un problema o insatisfacción con el servicio.

---

## R

**Rechazo** _(Rejection)_
: Decisión desfavorable sobre una solicitud o trámite.

**Reconsideración** _(Reconsideration)_
: Proceso de revisión de una solicitud rechazada. Debe solicitarse dentro de 30 días.

**Recuperación de Contraseña** _(Password Recovery)_
: Proceso para restablecer acceso cuando se olvida la contraseña.

**Registro** _(Registration)_
: Proceso de crear una cuenta nueva en el sistema.

**Renovación** _(Renewal)_
: Solicitud para extender la vigencia de un permiso o carnet que está próximo a vencer.

**Requisito** _(Requirement)_
: Condición o documento necesario para completar un trámite.

**Residencia** _(Residence)_
: Estatus legal que permite vivir permanente o temporalmente en Panamá.

**Resolución** _(Resolution)_
: Documento oficial que contiene la decisión sobre un trámite (aprobación o rechazo).

**Respaldo** _(Backup)_
: Copia de seguridad. Ver "Backup".

**Responsive** _(Responsive)_
: Diseño web que se adapta automáticamente a diferentes tamaños de pantalla (móvil, tablet, computadora).

**Reunificación Familiar** _(Family Reunification)_
: Proceso que permite a familiares de personas con estatus migratorio solicitar permiso para reunirse.

**Rol** _(Role)_
: Conjunto de permisos y capacidades asignadas a un usuario (Solicitante, Funcionario, Administrador).

**RUC** _(Registro Único de Contribuyentes)_
: Número de identificación fiscal en Panamá.

---

## S

**Sesión** _(Session)_
: Período de tiempo en que un usuario está activamente conectado al sistema. Expira tras 30 minutos de inactividad.

**Sistema** _(System)_
: Plataforma web completa para gestión de trámites migratorios.

**Servidor** _(Server)_
: Computadora que aloja y ejecuta el sistema, almacenando datos y procesando solicitudes.

**SMS** _(Short Message Service)_
: Servicio de mensajes de texto para notificaciones móviles.

**Soft Delete** _(Eliminación Lógica)_
: Método de eliminación donde el registro se marca como inactivo pero no se borra físicamente de la base de datos.

**Solicitud** _(Application/Request)_
: Trámite formal presentado ante el Servicio Nacional de Migración.

**Soporte Técnico** _(Technical Support)_
: Equipo encargado de resolver problemas técnicos del sistema.

**SSL** _(Secure Sockets Layer)_
: Protocolo de seguridad que encripta la comunicación entre navegador y servidor (indicado por HTTPS y candado en la URL).

**Status** _(Estado)_
: Ver "Estado del Trámite".

---

## T

**Tarjeta de Crédito** _(Credit Card)_
: Método de pago en línea aceptado (Visa, Mastercard, American Express).

**Tasa** _(Fee)_
: Monto a pagar por procesamiento de un trámite.

**Ticket** _(Ticket)_
: Número de caso asignado a una consulta o problema reportado a soporte.

**Tiempo de Espera** _(Wait Time)_
: Período estimado hasta que se complete una acción o se reciba respuesta.

**Timeout** _(Tiempo de Espera Agotado)_
: Cierre automático de sesión por inactividad prolongada.

**Timestamp** _(Marca Temporal)_
: Registro de fecha y hora exacta de un evento en el sistema.

**Traducción Oficial** _(Official Translation)_
: Traducción realizada por traductor público autorizado, requerida para documentos en idiomas no aceptados.

**Trámite** _(Procedure/Application)_
: Proceso administrativo completo desde solicitud hasta resolución final.

**Trata de Personas** _(Human Trafficking)_
: Causa humanitaria para PPSH relacionada con víctimas de tráfico y explotación humana.

**Tutorial** _(Tutorial)_
: Guía paso a paso que enseña a usar una función específica del sistema.

---

## U

**Upload** _(Carga)_
: Ver "Adjuntar". Proceso de subir archivos al sistema.

**URL** _(Uniform Resource Locator)_
: Dirección web del sistema (ej: https://tramites.migracion.gob.pa).

**Usuario** _(User)_
: Persona que utiliza el sistema. Puede ser solicitante, funcionario o administrador.

---

## V

**Validación** _(Validation)_
: Proceso de verificar que información o documentos cumplen con requisitos establecidos.

**Vigencia** _(Validity)_
: Período de tiempo durante el cual un documento o permiso es válido.

**Violencia de Género** _(Gender-Based Violence)_
: Causa humanitaria para PPSH relacionada con violencia basada en género o identidad sexual.

**Violencia Doméstica** _(Domestic Violence)_
: Causa humanitaria para PPSH relacionada con abuso intrafamiliar grave.

**Visa** _(Visa)_
: Autorización para ingresar y permanecer en Panamá por período determinado y con propósito específico.

---

## W

**Workflow** _(Flujo de Trabajo)_
: Secuencia estructurada de etapas por las que pasa un trámite. Cada etapa debe completarse antes de avanzar.

**WhatsApp** _(WhatsApp)_
: Aplicación de mensajería utilizada como canal de soporte (+507 6500-1234).
